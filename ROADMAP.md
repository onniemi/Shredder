# 路线图

目标不是“功能很多”，而是把 Windows 文件销毁这件事做到可信、快、稳定、可审计。

## v0.3 专业内核

- 可配置 I/O 策略真正贯穿 Core：缓冲区大小、进度节流、按卷/介质类型选择并发。
- 目录粉碎支持受控并发：同卷保守、跨卷并行，避免 HDD 机械寻道拖慢。
- Free-space wipe 尊重保留空间，支持取消后可靠清理临时文件。
- 审计报告增加吞吐量、失败分类、路径脱敏开关说明。
- 核心安全边界测试扩展到 ADS、重解析点、只读/隐藏/系统属性恢复。

## v0.4 Windows 集成

- 右键菜单安装/卸载 GUI 化。
- MSIX 或 Inno Setup 安装包。
- Windows 通知、任务栏进度、失败项导出。
- 更清楚的 SSD/NVMe 提示：默认 CryptoErase，明确物理擦除限制。

## v0.5 体验与可信发布

- i18n 资源化：简体中文、英文。
- 发布签名、SBOM、校验和。
- GitHub Release 自动构建 CLI/GUI 包。
- 基准测试报告：HDD、SATA SSD、NVMe、多小文件、多大文件。

## 以后再做

- ATA/NVMe Secure Erase / Sanitize 的受控入口。
- 卷影复制、文件历史记录、常见同步盘缓存的风险扫描。
- 企业策略模板与集中配置。
